# 術語定義表

用於檢查文件內術語一致性。當驗證規則指定「術語一致性」檢查時，會參照此檔案。

## 使用方式

- **standard**: 標準術語（文件中應使用此寫法）
- **variants**: 常見的變體寫法（應統一為標準術語）
